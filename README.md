The Button
==========

React/SVG rendering of [the button](http://reddit.com/r/thebutton)

Live Demo
---------
[here](http://dobrite.github.io/react-the-button/)

One terminal
------------

    npm install
    npm run start

Another terminal
----------------

    python -m SimpleHTTPServer

open browser to [localhost:8000](http://localhost:8000)
